📘 TP React – useState & useEffect

Ce projet est un TP d’apprentissage de React qui met en pratique les hooks principaux :

useState
useEffect

L’objectif est de comprendre la gestion de l’état et des effets dans une application React simple.

🚀 Fonctionnalités réalisées
🔹 1. Compteur simple
Incrémentation d’un nombre avec useState
🔹 2. Message dynamique
Changement de texte après un clic
🔹 3. Formulaire simple
Saisie d’un nom avec un formulaire contrôlé
🔹 4. Compteur avec effet (useEffect)
Affichage de la valeur du compteur dans la console
